Bootstrap allows you to style your background, text and border in 7 different colours.

## Text

| Class | Colour |
| :------ |:--- |
| `text-primary` | Primary colour |
| `text-secondary` | Secondary colour |
| `text-success` | Green |
| `text-danger` | Red |
| `text-warning` | Yellow |
| `text-info` | Blue |
| `text-muted` | Grey |
| `text-white` | White |
| `text-light` | Light-mode grey |
| `text-dark` | Dark-mode black |
